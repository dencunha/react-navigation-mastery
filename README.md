# React Navigation Mastery 🚀

## 🇺English Version

This practical project was developed to consolidate the fundamentals of navigation in Single Page Applications (SPA) using the latest version of **React Router (v7)**.

### Key Learnings & Features

During this module, I implemented a complete routing structure focused on organization and user experience:

- **Professional Setup**: Project initialization with Vite and TypeScript, including boilerplate cleanup and global style standardization.
- **Layout Routes (Shared Layouts)**: Implementation of a `<Layout />` component using `<Outlet />` to maintain persistent elements (Header/Footer) across multiple pages efficiently.
- **Dynamic Navigation & Hooks**:
    - `useParams`: Capturing dynamic URL parameters to display specific product details (`/details/:id`).
    - `useSearchParams`: Managing Query Params for filters and URL-persistent states.
    - `useNavigate`: Implementation of programmatic navigation, including the "go back" functionality (`-1`).
- **Error Handling**: Configuration of a fallback route (404 Not Found) to catch non-existent paths.

### Tech Stack
- React 18+
- TypeScript
- Vite
- React Router 7
- Modern CSS (Flexbox/Layouts)

## 🇧🇷 Versão em Português

Projeto prático desenvolvido para consolidar os fundamentos de navegação em aplicações Single Page (SPA) utilizando a versão mais recente do **React Router (v7)**.

### Aprendizados e Funcionalidades

Durante este módulo, implementei uma estrutura de rotas completa, focada em organização e experiência do usuário:

- **Configuração Profissional**: Setup com Vite e TypeScript, incluindo a limpeza de boilerplates e padronização de estilos globais.
- **Rotas de Layout (Shared Layouts)**: Implementação do componente `<Layout />` utilizando o `<Outlet />` para manter elementos fixos (Header/Footer) em múltiplas páginas de forma eficiente.
- **Navegação Dinâmica e Hooks**:
    - `useParams`: Captura de parâmetros dinâmicos na URL para exibição de detalhes de produtos (`/details/:id`).
    - `useSearchParams`: Manipulação de Query Params para filtros e estados persistentes na URL.
    - `useNavigate`: Implementação de navegação programática, incluindo a funcionalidade de voltar à página anterior (`-1`).
- **Tratamento de Erros**: Configuração de uma rota de fallback (404 Not Found) para capturar caminhos inexistentes.

### 🛠️ Tecnologias Utilizadas
- React 18+
- TypeScript
- Vite
- React Router 7
- CSS Moderno
