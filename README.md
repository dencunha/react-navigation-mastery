# React Navigation Mastery 🚀

Projeto prático desenvolvido para consolidar os fundamentos de navegação em aplicações Single Page (SPA) utilizando **React Router v7**.

## 🧠 Aprendizados e Funcionalidades

Durante este módulo, implementei uma estrutura de rotas completa, focada em organização e experiência do usuário:

- **Configuração Profissional**: Setup com Vite e TypeScript, incluindo a limpeza de boilerplates e padronização de estilos globais.
- **Rotas de Layout (Shared Layouts)**: Implementação do componente `<Layout />` utilizando o `<Outlet />` para manter elementos fixos (Header/Footer) em múltiplas páginas de forma eficiente.
- **Navegação Dinâmica e Hooks**:
    - `useParams`: Captura de parâmetros dinâmicos na URL para exibição de detalhes de produtos (`/details/:id`).
    - `useSearchParams`: Manipulação de Query Params para filtros e estados persistentes na URL.
    - `useNavigate`: Implementação de navegação programática, incluindo a funcionalidade de voltar à página anterior (`-1`).
- **Tratamento de Erros**: Configuração de uma rota de fallback (404 Not Found) para capturar caminhos inexistentes.

## 🛠️ Tecnologias Utilizadas
- React 18+
- TypeScript
- Vite
- React Router 7
- CSS Moderno
